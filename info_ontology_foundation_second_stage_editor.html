This editor is in an experimental state

When started the first-stage base ontology is loaded.

The user can constrain the information made available to the interfaces, that is the variable classed of the network that is being read from.

The interfaces have a fixed variable class called “transform”.

Control is assumed to have not only a variable class state but also differential_state latter is also indexed.

On pushing save the program generates the

  • all index structures

  • all base network matrices

  • all projection matrices for nodes, arcs and conversions

  • all selection matrices
